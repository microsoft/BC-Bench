Title: It is possible to select a Job in the Time Sheet even if the status of the Job is Planning, but later is not possible to post the Time Sheet via Job Journal as the Job does not have the Status Open.
Repro Steps:
1-Create a new Job with Status Planning (JOB0004) or use an existing one and change Status to Planning. 2-Open User setup and set a User as Time sheet administrator 3-Open Resources and set ALFREDO as Time sheet user, and from the ribbon select create time sheet and then select OK 4-Go to Time Sheet and select one of the created. Edit Type = Job and select the Job created in step 1 with Status = Planning, Job Task No and the hours per day for this resource: 5- Go to Manager Time Sheets and approve this one 6- Go to Job Journal and use the routine: Suggest Lines from Time Sheets... 7- Select the right filters to get the previous time sheet lines created: 8- The lines appear in the Job Journal: Try to post the Job Journal lines. ================== ACTUAL RESULTS ================== It is possible to select a Job in the Time Sheet even if the status of the Job is Planning, but later is not possible to post the Time Sheet via Job Journal as the Job does not have the Status Open. ================== EXPECTED RESULTS ================== This is confusing, so we should align this process and decide between: We should not allow to create Time Sheets if the Job Status <> Open We should allow to post Time Sheets with Status = Planning
Description:
It is possible to select a Job in the Time Sheet even if the status of the Job is Planning, but later is not possible to post the Time Sheet via Job Journal as the Job does not have the Status Open. This is confusing, so we should align this process and decide between: We should not allow to create Time Sheets if the Job Status <> Open We should allow to post Time Sheets with Status = Planning
