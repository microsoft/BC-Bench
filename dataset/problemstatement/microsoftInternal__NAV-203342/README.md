Title: [ALL-E] In the Belgium Version the Report Post Inventory Costs to G/L is using wrong parameters
Repro Steps:
Open BE environment. Go to General Ledger Setup. Use Personalize to add Enable Journal Templ. Name Mandatory. toggle. Journal Templ. Name Mandatory=yes Go to the Section Journal Templates in the General Ledger Setup and Apply Jnl. Template Name , Apply Jnl. Batch Name. Drill down into Apply Jnl. Batch Name. to check the No Series assigned. Starting No assigned is G05001. Once All setup. Go to Post Inventory Cost to G/L. You will find the Parameters already pre added based on the General Ledger setup we have completed. Change Journal Template Name and Journal Batch Name in the report parameters. The Journal Template BETAAL with Journal Batch name ALGEMEEN has a different number series that starts with the No. G04001. Click Preview. Check the report and you will find another No Series assigned (Incorrect). Actual Outcome: After checking the report, the system keeps respecting Journal Templates in the General Ledger Setup although we've changed the parameters in the report. The Document calculates from G05001 which is the default report done in the G/L Setup. Expected Outcome: The Document Should calculates from G04001 respecting the parameters added in the report.
