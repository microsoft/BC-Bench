# New method to add KeyInfo clause to SignedXml from X509 certificate

#### Summary
New method in the SignedXml codeunit to add KeyInfo clause to the signature from an X509 certificate.
Key info contains the **X509Data** element conforming to the [W3C XML signature syntax](https://www.w3.org/TR/xmldsig-core/#sec-X509Data).
